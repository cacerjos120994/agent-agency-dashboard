# Agent Agency Dashboard 🦊

A premium, AI-operated marketing agency mission control dashboard. Featuring a pixel-art office simulation, real-time agent status tracking, and automated mission logs.

## 🚀 Vision
Built for Jose's elite digital operations, this dashboard visualizes the workflow of 7 specialized AI agents working in harmony to deliver high-performance marketing results.

## 🛠 Tech Stack
- **Next.js 15+** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 📦 Features
- **Office View**: Pixel-art representation of agents at their desks.
- **Agent Intelligence**: 7 specialized roles (Research, Strategy, Copy, Planner, Analytics, Automation, Manager).
- **Live Logs**: Real-time activity feed of system operations.
- **Mission Metrics**: High-level KPIs and efficiency tracking.
- **Vercel Optimized**: Ready for instant deployment on Vercel Hobby.

## 🏃 Local Development
1. Clone the repo
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`
4. Open [http://localhost:3000](http://localhost:3000)

## 🚢 Deployment (Vercel)
1. Push this project to GitHub.
2. Import the project in [Vercel](https://vercel.com/new).
3. Click **Deploy**.

The project is pre-configured for the Vercel Hobby tier. No external databases or complex backends required for this demo version.
